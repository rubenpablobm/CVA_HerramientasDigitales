Tema 2: Software y Hardware

Tal vez hayas escuchado estas palabras antes. Todos los dispositivos digitales se conforman de software y hardware y el sistema operativo funciona como un puente que permite el funcionamiento del equipo.

El software es el conjunto de aplicaciones o programas que contienen las instrucciones o información de la computadora. El hardware son los componentes físicos de los que está hecha la computadora, las piezas que podemos tocar con nuestras manos.
