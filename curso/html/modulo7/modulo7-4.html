Tema 2 Introducción a la lógica de programación

Antes de iniciar a programar, se necesitas saber cómo piensa una computadora para poder decirle qué hacer. En el área de programación es muy importante utilizar la lógica, también las matemáticas para poder realizar cualquier operación.

Hay muchos elementos de lógica que se toman en consideración mientras uno diseña un programa. La página de woz-u nos cuenta de 5 elementos que son esenciales en saber en ningún orden en particular:

Haz clic en las imágenes para saber más de cada sistema operativo.

Inputs o Entradas

Son los datos que le damos a la computadora. Se pueden introducir con el teclado, el mouse o tocando la pantalla.

Outputs o Salidas

Es el resultado que regresa la computadora después de procesar los datos que le damos.

Operaciones Aritméticas

La computadora procesa los datos realizando operaciones matemáticas y lógicas.

Operaciones Condicionales

La computadora puede comprarar valores de manera lógica.

Loops o Cíclos

Los ciclos son instrucciones que se repiten una y otra vez, hasta que se cumpla una condición específica.
