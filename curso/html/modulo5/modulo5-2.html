Tema 1: Introducción a GIMP

Empezamos este módulo con ¿Qué es Gimp? Es un editor de imágenes multiplataforma disponible para GNU/Linux, macOS, Windows y más sistemas operativos. Es software gratuito, puedes cambiar su código fuente y distribuir tus cambios si lo quisieras hacer en algún momento. Como en Adobe Photoshop lo que busca gimp es ayudarte a que puedas editar o manipular fotografías.

En dicho programa se puede hacer gráficos y editar imágenes.

Audiovisualmente. (2021). Cómo Usar GIMP en 21 Minutos (Edición de Imágenes) - Guía para Principiantes, como Photoshop [YouTube Video]. In YouTube. https://www.youtube.com/watch?v=5G3Yq-w3hOI

Una vez que se familiarice con el uso de las herramientas y los iconos esenciales en la interfaz de usuario, definitivamente debería buscar más tutoriales para familiarizarte mejor con el programa y sus capacidades. Hay muchas teclas de acceso directo que tambien deberias checar para asi tener un flow de trabajo más continuo y sencillo. Algunos ejemplos son:

Acción Comando
Seleccionar todo Ctrl + A
Selector de color O
Mover M
Recortar y cambiar el tamaño Mayús + C
Escala Mayús + T
Nueva Imágen Ctrl + N
Guardar Imágen Ctrl + S
Guardar con un nombre nuevo Mayús + Ctrl + S

Finalmente solo juega y practica hasta que entiendas tus posibilidades con GIMP. Esto, para que lo puedas implementar a tus diferentes proyectos que se te puedan presentar.
