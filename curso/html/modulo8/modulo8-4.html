Tipos de datos

Python reconoce diferentes tipos de datos. Estos son algunos de ellos:

Tipo de dato Se escribe Descripción Ejemplos
Strings str Son valores de texto x="hola"
y="mundo"
Enteros int Son números enteros, positivos o negativos x = 5
y = 100
z = -20
Flotantes float Son números decimales, positivos o negativos x = 3.14
y = -5.2
Booleanos bool Solo pueden tener valor verdadero o falso x = true
y = false

Operadores aritméticos

La base de la programación son las matemáticas. Las computadoras pueden resolver problemas matemátivos de manera instantánea.

Para declarar expresiones aritméticas podemos utilizar los siguientes operadores:

Operador Ejemplo Resultado
Suma + 2 + 3 5
Resta – 50 - 15 35
Multiplicación * 2 * 10 20
División / 30 / 3 10
Módulo % 7 % 2 1
Exponente ** 2 ** 5 64
División entera // 9 // 4 2

Operadores relacionales

Los operadores relacionales nos indican si una expresión es verdadera o falsa. Las expresiones lógicas también son conocidas como condiciones.

Operador Explicación Ejemplo
Igual a == Regresa verdaderosi son iguales 1 == 1
(verdadero)
1 == 2
(falso)
No igual a !== Regresa verdadero si no son iguales 10 != 20
(verdadero)
10 != 10
(falso)
Mayor que > Regresa verdadero si el derecha es mayor que el de la izquierda 10 > 5
(verdadero)
5 > 10
(falso)
Menor que > Regresa verdadero si el derecha es menor que el de la izquierda 1 < 2
(verdadero)
5 < 3
(falso)

Condiciones if

Las condiciones if son bloques de código que se ejecutan solamente cuando una condición específica es verdadera.

Ejemplo 1:

if ( 5 == 5 ) : codigo

Ejemplo 2:

if ( 2 > 5 ) : codigo

La condición es lo que está dentro de los paréntesis. Si esa condición es verdadera, el código después de los dos puntos se corre. Si la condición es falsa, el código se ignora.
